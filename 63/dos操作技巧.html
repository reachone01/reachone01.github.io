在IE中进入DOS c:\windows\system32\cmd.exe
PageUp 重复第一条命令
PageDown 重复最后一条命令
↑↓ 在历史命令中按先后顺序逐条重新调用命令；
←→ 在命令中行中移动光标；
复制 1 在DOS窗口中复制：在标题栏右击---标记---选择---在标题栏右击---复制(or press enter)；
　 2 在其它程序中选择后---ctrl+c
粘贴 在标题栏右击---编辑---粘贴；
F7 显示历史命令列表
F9 输入命令号码（对应F7出现的新窗口）
alt+f9 全角和半角的切换
dos命令的参数用于扩充其功能，其引导的字符有：
- shutdown -f
/f sfc /scannow
p set p
d: chkdsk d:
esc 清除当前命令行
F7 显示保存在缓冲区中的命令历史记录
F8 搜索命令的历史记录，循环显示所有曾经执行过的命令；
F9 按编号选择命令，一般和F7配合使用
Alt+F7 清除所有曾经执行过的命令历史记录；
Home 快速移动光标到命令行的开关或结尾；
Ctrl+H 删除光标左边的一个字符；
Alt+F10 清除宏定义；
Tab 选择当前目录下的文件夹和文件（包括隐藏文件在内），按shift+tab还可以反选；

用批处理文件快速登录局域网中的某一台电脑
net use \\192.168.0.9 [password] /user:[username]
explorer \\192.168.0.9

备份和还原网络配置
@echo off
netsh interface ip set address name="本地连接" source=static addr=192.168.0.11 mask=255.255.255.0  //设置本地连接的IP地址和子网掩码
netsh interface ip set address name="本地连接" gateway=192.168.0.1 gwmetric=1  //设置网关
netsh interface ip set dns name="本地连接" source=static addr=192.168.0.8 register=primary  //设置主DNS
netsh interface ip add dns name="本地连接" addr=192.168.0.9 //设置备用DNS
netsh interface ip set wins name="本地连接" source=static addr=none    //设置wins

用regsvr32命令修复系统文件故障
修复浏览器故障
regsvr32 actxprxy.dll

防范网络脚本病毒（禁用FSO）
regsvr32 /u scrrun.dll

卸载ZIP功能
regsvr32 /u zipfldr.dll

卸载图片预览功能
regsvr32 /u thumbvw.dll

给WMP注册RM插件
下载并解压缩realmediasplitter.ax到windows\system32目录下
regsvr32 realmediasplitter.ax

修复WMP
regsvr32 jscript.dll

解决windows默认的文件关联混乱问题
regsvr32 /I shdocvw.dll
regsvr32 /I shell.dll
regsvr32 /I shdoc401.dll

解决windows添加/删除程序无法启动问题
regsvr32 mshtml.dll
regsvr32 /I shdocvw.dll
regsvr32 /I shell.dll

解决windows xp搜索功能故障问题
regsvr32 urlmon.dll
