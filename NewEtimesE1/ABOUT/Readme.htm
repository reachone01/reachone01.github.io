光盘使用简要说明

     本光盘与E-Times English (1)――《新E代英语（1）》――文字教材配套使用，旨在既能满足课堂教学的需要，又为学习者自主学习和向更深层次提高提供足够的空间和方便。以下就光盘内容结构及电脑配置要求作简要说明（详细使用说明请参见：FAQ）：

使用光盘要求的电脑配置

  • 本光盘内容采用网页形式编写。电脑硬件配置要求声卡，以求播放声音文件时有较好的效果。建议使用IE 4.0以上版本的浏览器，屏幕区域采用800×600以上的模式（采用IE 6，1024×768模式效果最佳）。

  • 推荐使用内嵌至IE浏览器的Microsoft Windows Media Player来播放各种声音文件，或者使用Winamp。 若使用QuickTime播放声音文件，可能会遇到不能同时看到文本文件的情形。这时请注意您的计算机的媒体播放器的首选设置。只有设置正确才能既看到文本文件，又能同时听取声音文件。如果一播放声音文件就看不到文本文件，一般情况下都是设置有问题，请找身边懂电脑的朋友或同事指点一下，肯定很快解决 。祝君好运。
